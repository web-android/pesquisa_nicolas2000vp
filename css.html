CSS

O que é?

CSS é uma linguagem que é utilizada para definir como os documentos escritos no formato HTML ou XML devem ser apresentados na página. Desse jeito definindo sua formatação, seu layout e entre outros, em outras palavras o HTML estrutura o conteúdo da página e o CSS é o que irá formatar.

Benefícios:

  • O CSS possibilita ter controles de vários layouts com apenas um arquivo CSS
  • Conseguir manter o mesmo layout em diversos navegadors
  • Aplicação de técnicas de maneira mais rápida e sofisticada
  • Pouco consumo de banda larga e melhor desempenho
