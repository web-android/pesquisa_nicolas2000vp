Sobre HTML

O que é HTML?

HTML é uma linguagem de marcação, ou seja, fornece instruções do que deve ser mostrado na página. Para isso, a linguagem usa um conjunto pré definido de elementos, que por sua vez, possui diversas tags com o objetivo de expressar um determinado conteúdo

Benefícios:

  • Melhora a performance da página da web e a navegação fica mais rapida
  • Ela é aceita em todos os dispositivos, inclusive dispositivos moveis
  • Ela é aceita em diversos Browsers
  • Armazenamento de dados offline
  • Permite utilizar canvas
